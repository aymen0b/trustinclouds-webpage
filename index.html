About the project

Description

The TRUSTINCloudS project investigates solutions for the major cybersecurity challenges of Cloud environments. It adapts traditional security mechanisms to address the specific threats of different Cloud types (IaaS, PaaS, …). TRUSTINCloudS specifies and implements new methodologies to strengthen Cloud security and to build a sovereign and trusted Cloud.

It investigates both the security of Cloud infrastructures and the security of outsourced data — two main scientific axes aimed at developing new methods and tools. Proofs of concept will be implemented within the shared infrastructure provided by the SILECS project.

Structure

Work Packages

WP0
Project Management
Coordination and administrative management of all project activities and partner institutions.
WP1
Risk Analysis & Threat Identification
Risk analysis and threat identification in highly dynamic multi-cloud infrastructures. Infrastructure auditing and control methods.
WP2
Infrastructure Protection
Research on incident detection, prevention, and mitigation strategies for Cloud infrastructure protection.
WP3
Sensitive Data Management
Identification of risks related to sensitive data management and usability. Evaluation of users' acceptability for more secure data usage.
WP4
Privacy Enhancing Technologies
Specification of a set of privacy enhancing technologies for protecting sensitive data in Cloud environments.
WP5
Implementations & Proofs of Concept
Concrete implementations and proof-of-concept validations for all investigated solutions across the project.
